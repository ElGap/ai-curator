# AI Curator Datasets

This directory contains built-in training datasets for AI Curator.

## Structure

Each dataset lives in its own folder with two files:

```
datasets/
├── starter-pack/
│   ├── metadata.json    # Dataset information (name, license, attribution, etc)
│   └── samples.json     # Training samples in AI Curator format
└── README.md
```

## Dataset Metadata (metadata.json)

### Required Fields

- **name** - Display name of the dataset
- **version** - Dataset version (semver)
- **description** - What this dataset contains
- **author** - Creator or organization (with URL)
- **copyright** - Copyright notice
- **license** - SPDX license identifier (e.g., "CC-BY-4.0", "MIT")
- **license_url** - URL to full license text
- **attribution** - How to cite/attribute this dataset
- **website** - Official dataset homepage
- **samples_file** - Path to samples file (usually "samples.json")
- **total_samples** - Number of training samples

### AI Curator Fields

- **dataset_id** - Target dataset ID in AI Curator (1 or 2)
- **dataset_name** - Name shown in AI Curator UI
- **goal_name** - Training goal display name
- **goal_samples** - Target sample count for this goal
- **default_category** - Default category tag
- **default_quality** - Default quality level (low/medium/high)

## Adding New Datasets

To add a new dataset:

1. Create a new folder: `datasets/your-dataset/`
2. Add `metadata.json` with all required fields
3. Add `samples.json` with training samples
4. Update this README.md to document it

## Samples Format

`samples.json` should be a JSON array of sample objects:

```json
[
  {
    "instruction": "Explain the offside rule",
    "input": "",
    "output": "The offside rule states that...",
    "system_prompt": "You are a football expert",
    "category": "football",
    "difficulty": "intermediate",
    "quality_rating": 5
  }
]
```

Each sample can include:

- `instruction` - The task/question (required)
- `input` - Additional context (optional)
- `output` - The expected response (required)
- `system_prompt` - System behavior prompt (optional)
- `category` - Category tag (optional)
- `difficulty` - Difficulty level (optional)
- `quality_rating` - Quality score 1-5 (optional)
